\# Security Policy



\## Supported Versions



| Version | Supported          |

| ------- | ------------------ |

| 5.0.x   | :white\_check\_mark: |

| < 5.0   | :x:                |



\## Reporting a Vulnerability



If you discover a security vulnerability, please open an issue or contact the maintainers directly.



Please do not disclose security issues publicly until they are resolved.
